# Franca IDL language support in Atom

## Introduction

This package provides support for
[FrancaIDL](https://github.com/franca/franca)

## Current Support

Currently this package supports highlighting for

* Comments
* Numerical constants
* String constants
* Keywords
  * struct
  * import
  * package
  * interface
